User Guide

Logging In

  1. From the home page click the link, "Sign In". You are redirected to a login page.
  2. Enter your e-mail address and password (case sensitive). If this is your first time using the system, follow the steps for signing up.
  3. Click the "Sign In" button. If you're e-mail and password match, you will enter to the user main page. Otherwise, you will be redirected to an error page. From this page you have the option to try again.

Creating An Account

If you already have an account, follow the steps for logging in. Otherwise, follow the following steps:
  1. From the home page click the link, "Sign Up". You are redirected to a registration form.
  2. Fill out all textfields in the form. Don't leave any fields blank
  3. When the form is filled out, click the "Register" button. If registration is successful, you will be redirected to the login page where you can login with your new account.
